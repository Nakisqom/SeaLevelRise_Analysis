# SeaLevelRise_Analysis

Berikut merupakan script MATLAB yang digunakan beserta fungsinya:
- ncsv.m           : konversi file dengan format NetCDF (.nc) menjadi (.csv)
- tidecheck.m      : memeriksa kelengkapan data pasang surut
- tiderunning.m    : master script untuk eksekusi utama pengolahan data pasang surut
- timeseries.m     : membuat grafik deret waktu untuk parameter oseanografi dan meteorologi
- trendanalysis.m  : melakukan analisis trend linear SLA
- validatesla.m    : validasi data pasang surut dengan data satelit altimetri
